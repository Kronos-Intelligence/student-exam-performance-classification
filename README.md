## 📖 Project Overview

This project demonstrates an end-to-end predictive analytics workflow using machine learning to classify whether students are likely to **pass or fail** an examination.

Two supervised learning models were developed and evaluated:

- Support Vector Machine (SVM)
- Logistic Regression

The project includes data preparation, exploratory data analysis, feature engineering, model training, model evaluation, feature importance analysis, and ethical considerations surrounding the use of predictive analytics in higher education.

---

## 🎯 Objectives

- Predict student examination outcomes
- Compare Support Vector Machine and Logistic Regression
- Evaluate model performance using multiple metrics

## 🏆 Results

The Support Vector Machine achieved the best overall performance.

| Metric | SVM | Logistic Regression |
|---------|----:|--------------------:|
| Accuracy | **83.95%** | 82.80% |
| Precision | **85.66%** | 84.55% |
| Recall | **85.20%** | 84.25% |
| F1 Score | **85.43%** | 84.40% |
| ROC-AUC | **91.61%** | 91.09% |

---

## 🔍 Key Findings

The strongest predictors of student success were:

- Study Hours
- Class Attendance
- Study Method
- Sleep Hours
- Sleep Quality

The analysis demonstrated that academic behaviors were more influential than demographic variables in predicting examination outcomes.
